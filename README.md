<div align="center">
    <h1>88x31-Button-Scraper</h1>
    <p>Scrapes popular websites for 88x31 buttons. Exports scraped elements to a .zip/.tar.gz compressed folder.</p>
</div>

## Dependencies

Install all of the required Python packages initially before running the script:
```bash
$ pip install -r ./requirements.txt
```

## Usage

```bash
$ python main.py
```
> Automatically detecting operating system for `.zip`/`.tar.gz` compression.
